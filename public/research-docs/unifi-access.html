UniFi Access — Prinzip in einem Satz

Jeder Zugangspunkt besteht aus Reader (liest Karte/NFC) + Hub (steuert Schloss) + Schloss (sperrt/öffnet). Gate Hub ersetzt Door Hub bei Toren. Access Ultra kombiniert Reader + Hub in einem Gerät.

PoE — Strom über Netzwerkkabel

Alle UniFi Access Geräte werden über ein einziges Ethernet-Kabel mit Strom und Daten versorgt. Kein separates Stromkabel nötig. Ein PoE-fähiger Switch (z. B. USW-Pro-8-PoE, ~180 €) versorgt alle Geräte zentral.

Controller — kein Cloud-Zwang

UniFi Access läuft lokal auf einer UniFi Dream Machine, Cloud Gateway oder dediziertem Server. Kein Cloud-Abo nötig. Remote-Zugriff optional via UniFi-Cloud oder eigenem VPN.

Alle Varianten im Detail

1. NFC-Karte — Tür öffnet kurz, schließt automatisch

Letzte Person geht raus, hält Karte an Außen-Reader → Tür öffnet für konfigurierbare Zeit (2–10 Sek.), Person tritt raus, Tür schließt und verriegelt automatisch (Fail-secure). Kein App-Tap, kein manuelles Sperren nötig.

Empfehlung einfachste tägliche Routine

2. Schlüsselschalter innen — 1 Drehen sperrt alles

Schlüsselschalter (Key Switch) neben der Ausgangstür. Drehen → Lockdown-Signal an Controller → alle Türen gesperrt. Person geht durch letzte Tür raus (die sich nach NFC-Tap schließt). Kein Handy nötig.

  • UniFi Access Rescue KeySwitch — 70 €
  • Anschluss: direkt an REX-Eingang → als Lockdown-Trigger konfigurieren
  • Beliebiger Schlüsselschalter (NO-Kontakt, ~15–25 €) alternativ möglich

3. Kippschalter + Status-LED (DIY)

Kippschalter innen (z.B. im Schaltkasten beim Ausgang): Umlegen → Lockdown. Status-LED zeigt den aktuellen Zustand:

  Kippschalter ──→ REX-Eingang Hub → Lockdown-Signal
  AUX2-Ausgang ──→ 12V → LED-Vorwiderstand → LED → GND
      Grün = normal    Rot = gesperrt

AUX2 kann als „Lockdown-Status"-Ausgang konfiguriert werden. Kosten: Schalter + LED + Widerstand ~5–15 €.

4. Zeitplan — automatisch sperren

In UniFi Access Schedules einstellen: z.B. täglich 18:30 Uhr → alle Türen auf „Locked". Kein manueller Eingriff nötig. Person muss nur vor 18:30 raus sein oder kennt den Code zum Öffnen nach Feierabend.

5. App — Lockdown-Button (mit Handy)

Alle anderen Türen im Lockdown — wie wieder öffnen?

Nach dem Lockdown sind alle nicht-REX-Türen gesperrt. Zum Wiederöffnen (z.B. nächsten Morgen):

  • Schlüsselschalter → Position 1 (Unlock All) — alle öffnen gleichzeitig
  • NFC-Karte an beliebigem Außenreader — öffnet diese Tür einzeln
  • App → einzelne Tür oder alle freigeben
  • Zeitplan → automatisch z.B. Mo–Fr 07:00 Uhr
Workflow komplett ohne App: Schalter Position 1 = alles offen · Position 2 = alles zu · REX-Knopf = Ausgang.
REX und Lockdown: Der REX-Eingang ist nach EN 13637 als Fluchtweg klassifiziert und öffnet immer — auch im Lockdown. Das ist gewollt und gesetzlich vorgeschrieben. Alle anderen Türen bleiben im Lockdown gesperrt und müssen mit NFC, App oder Schlüsselschalter wieder geöffnet werden.

Status-LED — Gebäude gesperrt oder offen?

Wie funktioniert die Status-LED?

Der Door Hub hat einen konfigurierbaren AUX-Ausgang. Dieser kann so eingestellt werden, dass er aktiv ist, wenn das System im Lockdown ist — und aus, wenn alles normal läuft. Daran hängt eine LED.

  Option A — Einfarb-LED:
  AUX2 NO ──→ 12V ──→ [Vorwiderstand ~470Ω]
                         ──→ [Rote LED] ──→ GND
  LED leuchtet = Lockdown aktiv
  LED aus      = Normal / offen

  Option B — Zweifarbige Ampel-LED:
  AUX2 → Rote LED   (Lockdown)
  AUX1 → Grüne LED  (Unlock All / normal offen)

Konfiguration in UniFi Access

Im Controller pro Hub konfigurierbar:

  • AUX2 → Modus: "Lockdown Status" → aktiv wenn Lockdown
  • AUX1 → Modus: "Door Unlocked Status" → aktiv wenn freigegeben
  • Spannungsversorgung LED: 12V über separates Netzteil oder Hub-intern
  • Vorwiderstand: 470 Ω bei 12V / rote LED (berechnen nach LED-Specs)
Ergebnis: Kleines LED-Panel neben dem Schlüsselschalter — auf einen Blick sichtbar ob alles gesperrt ist, bevor man das Gebäude verlässt.

Türöffnungszeit konfigurieren (Unlock Duration)

In UniFi Access pro Tür einstellbar: wie lange der Hub das Relay öffnet nach NFC-Tap. Standard: 3 Sek. Empfehlung Haupttür: 5–7 Sek. Nach Ablauf rastet Fail-secure Schloss automatisch ein.

App-Override — immer möglich

Da der Zustand im Controller (Software) liegt, kann die App jederzeit überschreiben — egal in welcher Position der physische Schalter steht. Schalter = Auslöser, App = Master.

Wichtige Eckdaten zum Hub-Dry-Contact

Lock-DRY ist potentialfreier Wechsler (NO + COM + NC) im UniFi Door Hub: max 30 V DC, 1 A. Bei 2 parallelen Schlössern an einem Netzteil können 2 A überschritten werden → Koppelrelais vor das eigentliche Schloss schalten.

Wann braucht man REX?

Bei Fail-secure Schlössern sind Türen auch von innen verriegelt. Damit man von innen jederzeit raus kann (Fluchtwegpflicht!), gibt es den grünen REX-Taster. Alternativ: mechanischer Panikdrücker oder Türbeschlag der die Falle trotzdem frei gibt.
